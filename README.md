GoLinks
===

A Simple GoLinks built in Next.js with Okta SSO

https://nextjs.org/docs


# Setup

```
git clone https://github.com/parabuzzle/golinks.git
cd golinks
cp .env.example .env
npm install
```

edit `.env`

setup the database (schema is found in `.db`)

# Running

```
npm run dev
```

# Running Production Server

```
npm run express
```
